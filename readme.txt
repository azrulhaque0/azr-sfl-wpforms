An extension of WPForms for Save for later functionality

== Description ==

Save for later plugin is used to add save for later button in step forms created by WP Forms. One can save its step using this button with AJAX

== Frequently Asked Questions ==

= What is SFL (Save for later) plugin ? =

Save for later plugin which is an extension to WP forms.

= Can we turn off save for later button without deactivating this plugin =

Yes you can turn off from plugin settings.
